📖 How to Play MathMart Arena

🎯 Game Objective

Answer math questions correctly to earn XP and coins. Level up, buy items, and climb the global leaderboard!

🚀 Getting Started

  1. Create Profile: Enter your name, choose avatar and color
  2. Select Challenge: Pick topic and difficulty level
  3. Answer Questions: 5 multiple-choice questions per challenge
  4. Earn Rewards: Gain XP and coins for correct answers

📚 Question Types (14 Types)

✓ Simplification
✓ Percentage
✓ Ratio
✓ Average
✓ Profit & Loss
✓ Simple Interest
✓ Time & Work
✓ Speed & Distance
✓ Age Problems
✓ Number Series
✓ Pattern Completion
✓ Logical Reasoning
✓ English Grammar
✓ Error Detection
NEW! Reasoning & English questions based on latest IBPS/SBI patterns:
  • Logical Reasoning: Blood relations, directions, coding-decoding, seating arrangement
  • English Grammar: Tenses, prepositions, articles, synonyms, antonyms
  • Error Detection: Sentence correction, grammar errors

⚡ Difficulty Levels

⚡ Basic
4 minutes • 50 XP • 30 Coins
🔥 Moderate
8 minutes • 100 XP • 60 Coins
💎 Advanced
16 minutes • 200 XP • 120 Coins

🏪 Shop & Items

  • 50+ Items: Buy with coins earned from challenges
  • 6 Rarities: Common, Uncommon, Rare, Epic, Legendary, Mythic
  • XP Bonuses: Some items give permanent XP boosts
  • Collection: Build your inventory and show off!

⭐ XP & Leveling

  • Earn XP by answering questions correctly
  • XP requirement increases with each level (quadratic scaling)
  • Higher levels unlock bragging rights on leaderboard
  • Track your progress in the dashboard

🏆 Leaderboard

Compete with players worldwide! Leaderboard shows top players ranked by:

  • Level: Primary ranking factor
  • XP: Tiebreaker for same level
  • Real-time: Updates instantly when you level up

⚠️ Rules & Regulations

  • Daily Limit: Maximum 50 questions per day (anti-cheat)
  • Timer: Answer within time limit or question auto-submits
  • Multiple Choice: Select one option from A, B, C, D
  • No Cheating: Fair play ensures fun for everyone
  • One Account: Create one profile per person
  • Penalties: Wrong answers reduce potential XP gain

💡 Pro Tips

  • Start with Basic difficulty to build confidence
  • Read questions carefully - pattern questions have tricks!
  • Use all available time - no rush for bonus
  • Save coins for high-rarity items with XP bonuses
  • Practice daily for best results in real exams
